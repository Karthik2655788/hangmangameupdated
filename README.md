# 🎮 Hangman Multi-Word Puzzle Game

[![HTML5](https://img.shields.io/badge/HTML5-%23E34F26.svg?style=flat-square&logo=html5&logoColor=white)](https://developer.mozilla.org/en-US/docs/Web/HTML)

[![JavaScript](https://img.shields.io/badge/JavaScript-%23F7DF1E.svg?style=flat-square&logo=javascript&logoColor=black)](https://developer.mozilla.org/en-US/docs/Web/JavaScript)

A **web-based Hangman-style puzzle game** where players guess letters to complete **multi-word phrases**.  
The game displays phrases in a **grid layout**, spaces between words are visible, and letters are filled in as you guess.

---

## 📝 Features

- Random **multi-word phrases**  
- **Grid layout** with visible spaces between words  
- **Letter input** for guessing  
- Correct guesses reveal all occurrences of the letter  
- **Win message** when the phrase is completed  
- **Game-style UI** with colored boxes and buttons  

---

## 🛠️ Technologies Used

- **HTML** – Structure of the game  
- **CSS** – Styling and responsive grid layout  
- **JavaScript** – Game logic: random phrase selection, letter checking, win detection

## ⚡ Setup Instructions

1. **Clone or download** this repository.  
2. Ensure the following files are in the same folder:
   - `index.html`  
   - `style.css`  
   - `script.js`  
3. Open `index.html` in any modern web browser.  

---

## 🎮 How to Play

1. The game will display a **grid with underscores** representing letters in a random phrase.  
2. Spaces between words are visible and do **not need to be guessed**.  
3. Enter a **single letter** in the input box and click **Guess**.  
4. Correct letters are revealed in all positions they appear.  
5. Incorrect guesses display a message.  
6. When all letters are guessed, a **win message** appears.
